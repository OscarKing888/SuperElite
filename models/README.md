# 模型权重目录

本目录用于存放 **爱好者水平**（PyIQA：NIMA + TOPIQ）所需的本地权重文件。

## 所需文件

| 文件 | 用途 | 说明 |
|------|------|------|
| `nima_ava.pth` | NIMA 美学评分 | 可从 NIMA/PyIQA 相关项目获取 |
| `cfanet_iaa_ava_res50-3cd62bb3.pth` | TOPIQ 质量评分 | TOPIQ/CFANet 预训练权重 |

若上述文件不存在，选择 **爱好者水平** 时预加载会失败。

## 推荐用法

- **詹姆斯水平**：使用 One-Align 模型，首次运行会自动从 HuggingFace 下载，**无需**本目录权重。
- **爱好者水平**：使用 NIMA + TOPIQ，需要将上述两个权重文件放入本目录后再使用。

若暂时没有 NIMA/TOPIQ 权重，请先在应用内选择 **詹姆斯水平** 进行评分。
